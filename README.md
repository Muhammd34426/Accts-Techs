# Accts-Techs Themed Website

A professional, responsive website for UON Corp, built with modern web technologies. This project showcases a corporate-themed site with multiple pages and interactive features.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Multiple Pages**: Includes home, about, contact, services, news, and more
- **Interactive Elements**: Sliders, animations, and contact forms
- **Modern UI**: Clean design with icons, images, and professional layout

## Pages Included

- Home (index.html)
- About Us (about-us.html)
- Career (career.html)
- Clients (clients.html)
- Contact (contact.html)
- FAQ (faq.html)
- Mission & Vision (mission-vision.html)
- News (various sidebar layouts)
- Our Offices (our-offices.html)
- Press Releases (press-releases.html)
- Prices (prices.html)
- Solutions (solutions01.html to solutions09.html)
- Team Members (team-members.html)
- Showcases (showcases.html)
- 404 Error Page (404-error.html)

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling with animations and responsive design
- **SCSS**: Preprocessed CSS for better organization
- **JavaScript**: Interactivity with libraries like jQuery, Bootstrap, Swiper, etc.
- **Fonts and Icons**: Font Awesome for icons
- **Images and Videos**: Media assets for visual content

## Local Development

1. Clone the repository:
   ```
   git clone https://github.com/Muhammd34426/Accts-Techs.git
   ```

2. Navigate to the project directory:
   ```
   cd Accts-Techs
   ```

3. Open `index.html` in your web browser to view the site locally.

## Deployment

This site is hosted on GitHub Pages for easy access.

**Live Link**: [https://muhammd34426.github.io/Accts-Techs/](https://muhammd34426.github.io/Accts-Techs/)

To enable GitHub Pages (if not already enabled):
1. Go to your repository on GitHub
2. Navigate to Settings > Pages
3. Select "Deploy from a branch" and choose `main` as the source branch
4. Save the changes

The site will be live at the above URL.
